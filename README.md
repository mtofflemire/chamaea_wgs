# Wrentit whole-genome analyses

Analysis scripts for the Wrentit (*Chamaea fasciata*) whole-genome resequencing project.

The numbered scripts in [`03_scripts`](03_scripts) cover variant filtering, population structure, phylogenetics, population-genomic summaries, demographic inference, genotype–environment association analyses, selection scans, and linkage-disequilibrium analyses.

## Notes

- Scripts are ordered approximately by analysis workflow.
- Large input datasets and analysis outputs are not included.
- Sample lists, metadata, reference files, and software environments must be supplied separately.
- Some scripts retain local workstation or computing-cluster paths and should be updated for the system on which they are run.
